# DeepTicker — 주식 & ETF 분석기

ETF 이름을 전달받아 공식 자료를 확인한 뒤, 상세 분석을 정적 HTML 리포트로 누적하는 프로젝트입니다.
별도 서버나 설치 없이 `index.html`을 열어 볼 수 있고 GitHub Pages로 같은 화면을 공유할 수 있습니다.

## 현재 범위

- ETF명/종목코드 검색
- 상품 구조, 비용, 지수 방법론, 주요 구성 종목
- 지수 성과와 변동성, 상승 동력, 위험, 시나리오
- 매수 전 체크리스트와 원문 출처
- 모바일 반응형 화면
- 주식 분석 영역은 요구사항 확정 전까지 준비 중 표시

첫 예시 리포트로 `KODEX 차이나휴머노이드로봇(0048K0)`이 들어 있습니다.

## 사용 방법

`index.html`을 더블클릭하면 브라우저에서 바로 열립니다.

ETF 분석 데이터는 `data/etfs.js`의 `window.ETF_ANALYSES` 배열에 하나씩 추가합니다.
가격, 보유 종목, 수익률처럼 바뀌는 숫자에는 반드시 기준일을 함께 기록합니다.

## 공유

공개 주소: https://dogok2.github.io/stock-etf-analyzer/

GitHub Pages는 `master` 브랜치의 `/ (root)`에서 배포합니다. 자세한 갱신 방법은 `DEPLOY.md`를 참고하세요.

## 데이터 원칙

1. 운용사 상품 페이지·투자설명서·지수사업자 방법론 등 1차 자료를 우선합니다.
2. 시세, 구성 종목, 성과는 기준일을 반드시 표시합니다.
3. 사실과 분석 의견을 구분합니다.
4. 특정 상품의 매수·매도 권유나 목표가격을 제시하지 않습니다.
5. 업데이트 때 이전 근거를 덮어쓰기보다 최신 기준일과 출처를 함께 갱신합니다.
